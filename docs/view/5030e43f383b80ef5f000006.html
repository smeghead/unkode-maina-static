[C] WinMainの1行目からウンコ

投稿者からのアピールポイント

データの連動っていうムチャな責任を他人に押し付ける前に、スタックメモリを食いつぶした責任を取ってもらおうか。

int WINAPI WinMain( HINSTANCE hInstance, /*略*/)
{
  // どれくらい必要になるかは、実行してみないと分からないので、
  // 大きめに取っておく。
  MyData arr[ 50000 ];

  // ↑ のうち、有効なデータの数。
  // arr の要素を変更したときは責任を持って連動させること！！
  int size = 0;

  return main_dialog(hInstance, arr, & size);
}

使い方ヒント: 「これは臭う」という行を見付けたら、各行のをクリックしてマーキングしておきましょう(要Twitter OAuth認証)
