[PHP] ループ処理？お前がそう思うんならそうなんだろう お前ん中ではな

投稿者からのアピールポイント

お前は何がしたいんだ。

$arr = array();
$r_count = 1;

foreach( $arr as $value ) {

  if( isnull( $value ) ) {
      break;
  }

  $value = $value + $r_count;

}

使い方ヒント: 「これは臭う」という行を見付けたら、各行のをクリックしてマーキングしておきましょう(要Twitter OAuth認証)
