[C] 伝説

投稿者からのアピールポイント

学生の頃、C言語の講義の筆記試験で、全く講義に出なかった人間が一列に並び、 先頭のできる学生の答案を順に写して提出した伝説の答案です。 彼らは後日呼び出され、

「行末のコレは何かね？」

「Jです！( ｰ`дｰ´)ｷﾘｯ」

「はい不合格」

となりました。

本件はセミコロン-J事件、彼らはj-menと名付けられ伝説となりました。

void main(void) {
    int x[] = {4, 6, 2, 1, 0, 9, 8, 7, 5, 3}j
    int n = 10j

    printf("ソート中:\n")j
    quickSort(x, 0, n - 1)j
}

   :
