[C] おまじないのおぼえかた

投稿者からのアピールポイント

ウンコードの趣旨とは完全に違いますが、自分のC言語の講義2回目のプリントをもとにしたネタを紹介します。
ちなみに、stdioがstd+ioと気付いたのはだいぶ後になってからです。(今でもstdの意味を尋ねられると答えられないのですが……。)

#include <stdio.h>
/* studioではなく、stdio。「studioからuを抜く」と覚える。 */

int main(void)
{
	printf("Hello world!");
	return 0;
}
