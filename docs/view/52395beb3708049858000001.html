[C++] 一社相伝の伝統を受け継いだ秘伝のソース

投稿者からのアピールポイント

Setterの役割なのにIsPrintCan・・・
全く役割が違うのにオーバーロード・・・
挙句の果てに、コメントが[in] or [out]しか違いがない・・・
この関数を呼んでいる箇所にも もちろん コメントなんてありません！

//印刷設定可[in]
void IsPrintCan(BOOL bPrintCan)
{
    //メンバ変数へ代入
};

//印刷設定可[out]
bool IsPrintCan()
{
    //メンバ変数から判定してbool値を返す
};
