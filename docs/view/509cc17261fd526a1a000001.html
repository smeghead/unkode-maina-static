[Objective-C] 意味わかって書いているかい

投稿者からのアピールポイント

クラッシュが絶えないアプリのプロジェクトに放り込まれた時。 参照カウントの扱いが間違いだらけで、"alloc] init] retain]" でプロジェクト内検索してみたらいっぱい引っかかった。うち１カ所がこんなだった。

@interface Hogehoge {
	NSString* text;
}
@end

@implementation Hogehoge
- (void)hoge {

	text = [[[[NSString alloc] init] retain] autorelease];

}
