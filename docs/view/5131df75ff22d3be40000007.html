[PHP] 拡張子は合っているけど

投稿者からのアピールポイント

テンプレートとしてPHPファイルを用意しておいて、それをドライバで読み取って出力する、というフレームワークなのですが、一部にはこんなファイルも存在します。

<!DOCTYPE html>
<html>
  <head>
  ....
  </head>
  <body>
    <h1>タイトル</h1>
    <div>
      <!-- ひたすらHTMLタグ -->
    </div>
  </body>
</html>
<!-- 結局、最後まで<?php ... ?>や <?= ... ?> は一切出現しなかった -->

使い方ヒント: 「これは臭う」という行を見付けたら、各行のをクリックしてマーキングしておきましょう(要Twitter OAuth認証)
