[VB.net] "TRUE"に完全一致しなければ処理しません！

投稿者からのアピールポイント

以前コードレビューを頼まれた中に入っていたコード（レプリカw）。

なんといっても文字列Booleanで関数利用者が仕様を知っているかどうかを厳密にチェックする、厳密規定派プログラミングスタイルが硬派です。

そして、戻り値も徹底した文字列主義で利用者は最後まで安心できません。徹頭徹尾、俺の流儀に従ってもらうという姿勢が強く表れています。

    Public Function ModeKirikae(ByVal flag As String) As String

        If flag = "TRUE" Then
            'コントロールの状態変更処理(省略)
            Return "TRUE"
        Else
            Return "FALSE"
        End If

    End Function
