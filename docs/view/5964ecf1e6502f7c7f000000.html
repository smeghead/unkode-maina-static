[C] この配列、狭いッ！！

投稿者からのアピールポイント

大学の同級生がこんなウンコード書いてコアダンプになって、僕にヘルプを求めてきました。 そこは配列の大きさを書くんだよ！

さらに欲を言えば、return 0欲しい！

実際はもっとコードがあるのですが、インデントや空白はお察しください。

#include<stdio.h>
#include<stdlib.h>

int main(void){
    int A[0][0],A1[0][1],A3[1][1];
    double A2[1][0];

    A[0][0]=0,A1[0][1]=1,A2[1][0]=-0.1*0.1,A3[1][1]=0;

exit(EXIT_SUCCESS);
}
