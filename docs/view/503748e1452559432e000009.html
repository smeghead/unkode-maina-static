[その他] [CSS] 緑なのはよくわかった。で緑って何？

投稿者からのアピールポイント

昔よく見かけた。お願いだから論理的な名前をつけてくだしあ

ウンコなコメント例もつけてみました。

.green {
    color: green;
}

/* さらにひどいのになるとこうなる */
.red {
/*
    2012/08/24 sk44_ 変更
    color: red;
*/
    color: orange;
}

使い方ヒント: 「これは臭う」という行を見付けたら、各行のをクリックしてマーキングしておきましょう(要Twitter OAuth認証)
