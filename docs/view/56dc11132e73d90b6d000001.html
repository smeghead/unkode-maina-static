[C] 便利なコメントアウト

投稿者からのアピールポイント

デバッグのためにこんなコメントアウトの仕方をしているのは、多分僕だけだと信じたい。
※分かりやすくするため、あえて同じ名前の関数を書いています。

#include <stdio.h>

// 有効
int main() {
    int i;
    
    for (i = 0; i < 100; i++) {
//      /*
        printf("Hello, world!");
        //*/
    }
}

// 無効
int main() {
    int i;
    
    for (i = 0; i < 100; i++) {
        /*
        printf("Hello, world!");
        //*/
    }
}

使い方ヒント: 「これは臭う」という行を見付けたら、各行のをクリックしてマーキングしておきましょう(要Twitter OAuth認証)
