[Objective-C] 意地でも放さない（掴まえもしないけど

投稿者からのアピールポイント

リリース済みのアプリのプロジェクトにこんなコードが入ってるのさっき見つけた。 前任者がもうここにいないか外注さんらしいので意図は解らない。

- (id)retain {
	return self;
}

- (oneway void)release {
}

- (id)autorelease {
	return self;
}

使い方ヒント: 「これは臭う」という行を見付けたら、各行のをクリックしてマーキングしておきましょう(要Twitter OAuth認証)
