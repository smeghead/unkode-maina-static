[Java] もうコメント書くなお前

投稿者からのアピールポイント

改修案件でぶち当たったソース マジでこんなのに遭遇することってあったんだなぁ・・・ 他にもろくに追記されていないjavadocがゾロゾロ・・・ 頼むから、javadoc書くなら最後まで責任持ってくれよ・・・

/**
  *
  *   〇〇の値を取得する
  *
  */
public void getFoo(int foo){
    this.foo = foo;
}

使い方ヒント: 「これは臭う」という行を見付けたら、各行のをクリックしてマーキングしておきましょう(要Twitter OAuth認証)
