投稿者からのアピールポイント

本気で。@use jQuery でお楽しみ下さい。

$(function(){

    hogehoge(param);

});

param = {
  unko: 1,
  unko2: 100
};

// liveにすると動いた
$(".Act_Button").live("click",function(){
  // 処理
})

使い方ヒント: 「これは臭う」という行を見付けたら、各行のをクリックしてマーキングしておきましょう(要Twitter OAuth認証)
