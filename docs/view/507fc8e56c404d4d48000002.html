[PHP] foreach大好き

投稿者からのアピールポイント

とりあえず、なんでも回しとけ！

if (count($account->getStoreList()) == 1)
{
	foreach ($account->getStoreList() as $store)
	{
		$this->_redirect('/shop/detail/'. $store->getId());
	}
}

使い方ヒント: 「これは臭う」という行を見付けたら、各行のをクリックしてマーキングしておきましょう(要Twitter OAuth認証)
