[PHP] 斬新なうんこ。配列をうんこが使うとこうなる

投稿者からのアピールポイント

絶句...


一度脳みそ取り出してスチールウールでガリガリ洗浄して戻したらいいんじゃないかと思う。
WordPress+Smartyとかで構築する前に基礎を学ぶべきうんこソース。

キレそう

フォームからPOSTで渡ってきた数値データをキーにするという発想ができたのにどうしてこうなったんだろうか・・・。

//○○配列
$hoge[0] = 'hoge';
$hoge[1] = 'hogee';
$hoge[2] = 'hogeee';
...
$hoge[50] = 'hogeeee';

$string = "p_hoge：";
for($i=0; $i<=50; $i++){
  if($_POST['p_hoge'] == $i){
    $string .= $hoge[$i];
  }
}
echo $string;

使い方ヒント: 「これは臭う」という行を見付けたら、各行のをクリックしてマーキングしておきましょう(要Twitter OAuth認証)
