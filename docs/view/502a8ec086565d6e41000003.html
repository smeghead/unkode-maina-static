[PHP] forとifの使い方覚えました！

投稿者からのアピールポイント

うれしくて使いたかったんだと思います。 ・・・怒ったので、あとでベタに書き直しました。

for ($i = 1; $i <= 100; $i++) {
    if ($i == 1) {
        $d[$i] = "hoge";
    }

    if ($i == 2) {
        $d[$i] = "piyo";
    }

    /* 3 ... 99 */

    if ($i == 100) {
        $d[$i] = "fuga";
    }
}

使い方ヒント: 「これは臭う」という行を見付けたら、各行のをクリックしてマーキングしておきましょう(要Twitter OAuth認証)
