[C] やりたいことはわかる、けど。

投稿者からのアピールポイント

※コードはもちろん原文のままではなく構造だけを表しています かなり昔、自分の担当箇所を引き継いた新人ＰＧが書いて、コードレビュー兼ＭＴＧで大発表したコード。 やりたいことはなんとなくわかる。変数 a, b, c の内容を加工して、結果を呼び出し元で使いたかったんだよね。 でも、どうしてその通りに動いていない（ビルドだけは通る）ものをレビューに持ってきちゃったんだい。それと、レビュー前に本線コミットしちゃってたんだい…。

int func( int a , int b , int c)｛
	a = 0;
	b = 1;
	c = 2;
	return ( a , b , c );
｝

使い方ヒント: 「これは臭う」という行を見付けたら、各行のをクリックしてマーキングしておきましょう(要Twitter OAuth認証)
