[VBA] そいつは行じゃねえ。

投稿者からのアピールポイント

マルチバイトな変数にインテリセンスできることの方が個人的には驚いた記憶が。

Sub 関数()
    Dim シート1 As Worksheet
    Dim 行 As Integer
    Set シート1 = ThisWorkbook.Worksheets(1)
    For 行 = 1 To 10
        MsgBox (シート1.Cells(1, 行))
    Next
End Sub

使い方ヒント: 「これは臭う」という行を見付けたら、各行のをクリックしてマーキングしておきましょう(要Twitter OAuth認証)
