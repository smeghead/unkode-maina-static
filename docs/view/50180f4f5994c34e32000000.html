[C#] 名前付けをしなさい。

投稿者からのアピールポイント

デザイナで調子に乗ってコントロール配置すると、自動生成された名前をそのまま使ってしまいがち。 そしてバグが出たときにうーんーうーんー悩むことになります。

void button1_Click(object sender,EventArgs e){
      button2.Text = "hoge";
}

使い方ヒント: 「これは臭う」という行を見付けたら、各行のをクリックしてマーキングしておきましょう(要Twitter OAuth認証)
