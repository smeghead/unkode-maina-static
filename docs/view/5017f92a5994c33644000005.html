[Java] 定数地獄

投稿者からのアピールポイント

ONEが2になることはこの先あるのでしょうか。。。意味のある名前に数値をつけるならまだしも。。。

public static final int ZERO = 0;
public static final int ONE = 1;
public static final int TWO = 2;
...
...
public static final int ONE_HUNDRED = 100;
