[PHP] 納品先からの要望

投稿者からのアピールポイント

PHPのウンコードではないが。たまたまPHP案件だったのでPHPのウンコードということで。

納品受入拒否理由： includeは、ファイルオープンのオーバヘッドがかかるので2回まで。

当時の私は、社会人なりたてのペーペーで。 リーダーに意見具申しても「お客さんの要望だから」と...。 ぅぅ...。

10年前とか書いてましたが。PEARがあったのは間違いなくて...でも出た当初で...4~5年前ですかね?

-- いろんなphp
<?php
  include("all_in_one.php");
  // 普通のコード。略。

-- all_in_one.php
ありとあらゆるincludeを展開したファイル。
何万行にも及ぶ。 正直開きたくもない。
あまりにも長すぎるし、覚えていない。
