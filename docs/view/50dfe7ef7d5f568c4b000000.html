[JavaScript] べた褒めするコード

投稿者からのアピールポイント

Unicodeだから一応動く(

<script>
var γ = 10; 
var Δ = 5;
alert(γ*Δ);
var γ = "イケメン";
var ζ = "♂"; 
for(ξ=0;ξ<Δ;ξ++){
alert(γ+ζ);
}
</script>

使い方ヒント: 「これは臭う」という行を見付けたら、各行のをクリックしてマーキングしておきましょう(要Twitter OAuth認証)
