[HTML] divとjavascriptだけでなんでもできるよ

投稿者からのアピールポイント

以前関わったサイトが外注から内製に変わって、久々にサイト見てきたらコードが完全に別物になってしまった

ちなみにgeneral.cssは使われてないけど、中身もかなりやばいことになってる

<head>
  <title>Hoge</title>
  <link href="css/resetCss.css" />
  <link href="css/general.css" />
  <script src="scripts/jquery.js"></script>
  <script src="scripts/jquery.lightbox.js"></script>
  <link href="css/jquery.lightbox.css" media="screen" />
  <script></script>
</head>
<body>
  <div style="大量のinline style" onclick="javascript:location.href='index.php'">
    ここにロゴ
  </div>
  <div style="大量のinline style">
    <div style="大量の（略">

        ここにサイト名
      </div>
    </div>
    <script>
      function checkValidation()
      {
        /* いろいろ */
        document.myForm.submit();
      }
    </script>
    <form action="hoge.php" method="post" name="myForm">
      <div style="(ry">
        <div style="(ry">
          <div style="(ry">
            <div style="(ry">

          <!-- 送信ボタン -->

        </div>
