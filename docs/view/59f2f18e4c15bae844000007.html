[JavaScript] なぜswitchしてしまったのか

投稿者からのアピールポイント

きっと初心者さんだったんだよ……

for(let i = 1; i < 0; i++) {
  switch(i) {
    case: 1
    case: 3
    case: 5
    case: 7
    case: 9
      console.log('奇数');
      break;
    case: 2
    case: 4
    case: 6
    case: 8
    case: 10
      console.log('偶数');
      break;
    default:
      console.err('数字が不正です。');
      break;
  }
}

使い方ヒント: 「これは臭う」という行を見付けたら、各行のをクリックしてマーキングしておきましょう(要Twitter OAuth認証)
