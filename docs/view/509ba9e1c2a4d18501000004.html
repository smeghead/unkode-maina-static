投稿者からのアピールポイント

2000 年頃に見たコードだがある意味彼は KVS を先取りしていたのかもしれません。

// 100件のデータを取得
for(int i=0; i<100; i++){
  stmt = con.prepareStatement("SELECT * FROM HOGE WHERE ID=?");
  stmt.setInt(1, i);
  rs = stmt.executeQuery();
  if(rs.next()){
    list.add(new Hoge(rs));
  }
  rs.close();
  stmt.close();
}
